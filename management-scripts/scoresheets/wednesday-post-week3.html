Quick Guide: How to Fill Out This Scoresheet
1. RECORD SCORES: Write the match scores in the empty SET 1, SET 2, and TB boxes (e.g. 6-4, 4-6, 10-7).
2. LINE POINTS: Write the calculated points for each team in the Home and Away boxes under the PTS column.
3. SUBSTITUTES & LEVEL 3: Circle the Level 3 players who play at each line. If a sub plays, cross out the prefilled player name and write the sub's name clearly.
4. CALCULATE LINE POINTS (Max 3 per Line):
• 1 Point for participation (if line is played).
• 1 Point for each set won (maximum of 2 set points).
Example: Winning 6-4, 6-4 = 3 pts. Losing 4-6, 4-6 = 1 pt.
5. Match Totals: Sum all 4 line points (max 12 pts total) and write them in the TOTAL POINTS box. Both Captains must sign.
